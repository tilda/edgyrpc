# edgyrpc
Microsoft Edge (Chromium-based) Discord RPC.

# Installing
Requirements:
- Python 3.6+
- `pipenv`
- A functioning brain

1. Clone the repository: `git clone https://github.com/tilda/edgyrpc`
2. Install dependencies: `pipenv install`
3. Run the program: `pipenv run rpc`
4. Profit!
